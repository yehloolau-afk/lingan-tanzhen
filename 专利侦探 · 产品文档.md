灵感探针 · 产品文档

## 是什么

输入一句中文描述，灵感探针帮你扫描国内外专利数据库，告诉你这个想法是否已经有人占坑，以及该怎么把它变成一个更好的产品方向。

> 面向创业者、产品经理、研究人员——不懂专利术语也能用。

---

## 核心功能

### 功能 1：检索 idea 可行性

**输入**：一句中文描述，比如"有没有某种测绘工具"

**执行顺序**：
1. LLM 把描述翻译成中英文检索关键词
2. 优先检索国内专利（Lens.org 过滤 CN 文献 / CNIPA 数据）
3. 再检索国际专利（USPTO 美国 + EPO 欧洲）
4. LLM 综合分析，生成报告

### 功能 2：结果处理与反馈

**情况 A：已有相似专利**
- 列出最相似的 1-5 项
- 把专利术语翻译成大白话（每项 2-3 句话）
- 说明与你的想法的关键区别

**情况 B：尚无相似专利**
- 评估创新性（1-10 分）
- 如果创新性不足：给出 3-5 条优化建议
- 如果创新性够：给出申请专利的核心要点建议

## 数据来源

| 来源 | 覆盖范围 | 费用 | 说明 |
|------|---------|------|------|
| Lens.org API | 全球 95+ 专利局，含 CNIPA | 免费（需注册） | 主力数据源 |
| USPTO PatentsView | 美国专利 | 完全免费无需注册 | 备用/补充 |
| DeepSeek API | LLM 分析层 | 按量付费 | 关键词提取 + 报告生成 |

**检索策略**：先 CN（中国专利），再 US/EP（国际专利），优先展示国内结果。

## 技术架构

```
用户输入（中文描述）
    ↓
DeepSeek：提取中英文关键词
    ↓
并行搜索（3 路）
  ├── Lens.org / jurisdiction=CN   → 国内专利
  ├── Lens.org / jurisdiction=US   → 美国专利
  └── USPTO PatentsView             → 美国专利（备用）
    ↓
DeepSeek：分析相似度 → 生成中文报告
    ↓
Web 界面展示
```

**技术栈**：Python + FastAPI + DeepSeek API + Lens.org API
**前端**：单页 HTML，无框架依赖
**运行方式**：本地运行，`python app.py` 自动打开浏览器

## 快速启动

### 1. 安装依赖
```bash
cd 专利侦探
pip install -r requirements.txt
```

### 2. 配置 API Key
```bash
cp .env.example .env
# 编辑 .env，填入你的 Claude API Key
# 可选：填入 Lens.org API Key（免费注册，用于中国专利搜索）
```

### 3. 运行
```bash
python app.py
# 自动打开浏览器 http://localhost:8000
```

## API Key 获取

| Key | 获取地址 | 是否必需 |
|-----|---------|---------|
| `DEEPSEEK_API_KEY` | platform.deepseek.com | 必需 |
| `LENS_API_KEY` | lens.org 注册后申请 | 强烈建议（无此 Key 则跳过中国专利搜索） |

Lens.org 注册免费，API 申请填写用途选"Academic / Research"即可，通常 1 天内审批。
